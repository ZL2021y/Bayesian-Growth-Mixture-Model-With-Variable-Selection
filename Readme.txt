Application to CAMP data
The file "Run ALL Models.R" is the R codes running all models discussed in the manuscript, that is Blasso, Horseshoe, and SSVS priors. The functions used in "Run ALL Models.R" should all be placed in the same folders, which include "Blasso GMM.R", "Horseshoe GMM.R", "SSVS GMM.R" and "PG.R"
